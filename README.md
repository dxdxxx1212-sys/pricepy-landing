# ПрайсПрицеп — лендинг с квизом и каталогом

Готовый одностраничный лендинг (HTML + CSS + JS, без зависимостей и сборки).
Открывается двойным кликом по `index.html`, хостится на любом статичном хостинге
(Netlify, GitHub Pages, Vercel, обычный хостинг, Beget и т.д.).

## Что внутри
- **Шапка** с навигацией, телефоном и CTA (адаптив + бургер-меню).
- **Hero** с заголовком, преимуществами и карточкой оффера.
- **Лента-тикер** с ключевыми триггерами доверия.
- **Блок «Почему мы»** — 6 преимуществ.
- **Каталог** — карточки товаров со скидками (данные в массиве `PRODUCTS`).
- **Квиз из 7 шагов** — прогресс-бар, «эксперт-ассистент», одиночный/множественный
  выбор, авто-переход, финал со сбором контактов и экраном благодарности + сводкой.
- **Отзывы**, **логотипы партнёров**, **FAQ-аккордеон**.
- **CTA-форма** внизу с выбором мессенджера и согласием на обработку данных.
- **Футер** с реквизитами и модалкой «Политика конфиденциальности».
- Плавающая кнопка, тосты, валидация форм.

## Как править контент
Всё структурировано в начале `<script>` в `index.html`:
- `PRODUCTS` — товары каталога (название, цена, скидка, иконка).
- `QUIZ` — вопросы квиза (`single` / `multi` / `contact`).
- `FAQ` — вопросы-ответы.
- Тексты hero/преимуществ/отзывов/футера — прямо в HTML-разметке.
- Цвета и шрифты — в блоке `:root` (CSS-переменные) вверху файла.

## Куда уходят заявки
Функция `sendLead(data)` в `index.html` сейчас выводит заявку в консоль и
показывает уведомление. Внутри неё — закомментированные примеры подключения:
1. POST на свой сервер (`/api/lead`);
2. напрямую в **Telegram-бот** (`sendMessage` через токен и chat_id);
3. внешний сервис форм.
Раскомментируйте нужный вариант и подставьте свои данные.

## Важно перед публикацией
Замените демонстрационные тексты, цены, картинки (сейчас эмодзи-плейсхолдеры),
телефон, e-mail и **реквизиты компании** в футере на свои.
